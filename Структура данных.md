# Структура данных базы данных фасада "стойка-ригель"

## Обзор
База данных описывает конструкцию фасада многоэтажного дома с 4 плоскостями.
Название файла соответствует названию плоскости (например, `_Шаблоны(5-1)-(5-6).xlsx`).

## Особенности конструкции

### 1. Логика вентстворок (ВС)
**ВС** — обозначение вентстворки.

Для рядов с типом заполнения **ВС** в таблице `РядыРигелей{XXX}` указана высота ряда.

**Заполнение НВС (Ниже Вент Створки)** делится ригелем:
- Выше ригеля — заполнение **ВС** (вентстворка)
- Ниже ригеля — заполнение **ПП**

Пример: на втором этаже плоскости (5-1)-(5-6) от второго до четвёртого ряда ригеля идёт заполнение ПП, но в месте заполнения ВС:
- От 2-го до 3-го ряда ригеля — заполнение **ПП**
- От 3-го до 4-го ряда ригеля — заполнение **ВС**

Другие заполнения не делятся ригелем по оси ВС.

### 6. Алгоритм расчета размеров и размещения заполнений

#### 6.1. Определение высотных зон (рядов ригелей)
1. Из таблицы `РядыРигелей{XXX}` выбрать все строки для заданного этажа.
2. Количество строк = количество рядов ригелей на этаже.
3. Для каждого ряда вычислить высоту установки ригеля от низа стойки:
   ```
   H_ригеля_от_низа = Установка - НизСтойки
   ```
4. Для расчета высоты верхнего заполнения текущего этажа потребуется высота установки первого ригеля следующего этажа (или верха стойки, если это последний этаж).

#### 6.2. Расчет высоты заполнений
1. **Базовое правило:** Высота зоны заполнения равна разнице высот соседних ригелей:
   ```
   H_зоны = H_ригеля[i+1] - H_ригеля[i]
   ```

2. **Правило "через ряд":** Большинство заполнений (кроме ВС) идут через ряд, поэтому их фактическая высота рассчитывается как сумма двух соседних зон:
   ```
   H_заполнения = H_зоны[i] + H_зоны[i+1] = H_ригеля[i+2] - H_ригеля[i]
   ```

3. **Исключение — вентстворки (ВС):**
   - Заполнение типа **ВС** не идет через ряд.
   - Ригель делит составное заполнение **НВС** на две части:
     - Ниже ригеля: заполнение **ПП** (высота = H_ригеля_ВС - H_ригеля_нижний)
     - Выше ригеля: заполнение **ВС** (высота = H_ригеля_верхний - H_ригеля_ВС)
   - В спецификации НВС представляется как два отдельных заполнения: ПП и ВС.

#### 6.3. Расчет ширины заполнений
1. Из таблицы `РядыРигелей{XXX}` получить значение `ТипРядовСтоек` для данного ряда.
2. По этому типу выбрать соответствующие строки из таблицы `РядыСтоек{XXX}`.
3. Рассчитать расстояния между соседними стойками:
   ```
   L_пролета = Установка[стойка_i+1] - Установка[стойка_i]
   ```
4. Ширина заполнения рассчитывается по формуле:
   ```
   W_заполнения = L_пролета - 25 мм
   ```

#### 6.4. Размещение заполнений в проемы
1. Для каждого ряда ригелей (из `РядыРигелей{XXX}`):
   - Получить тип заполнения из столбца `ТипРядовЗаполнений`.
   - По типу и `ТипРядовСтоек` получить марку заполнения из таблицы `РядыЗаполнений{XXX}`.

2. **Заполнения типа "Межэтажка":**
   - Перекрывают всё изделие целиком (от левой до правой границы изделия).
   - Ширина = сумма всех пролетов в пределах изделия − 25 мм (или рассчитывается индивидуально по крайним стойкам изделия).

3. **Заполнения типов "Зап1", "Зап2", "ВС":**
   - Размещаются в каждый пролет между стойками отдельно.
   - Каждое заполнение имеет свою ширину в зависимости от конкретного пролета.

4. **Заполнения типа "СН":**
   - Располагаются между изделиями (в стыке изделий).

#### 6.5. Формирование марки заполнения
Марка заполнения формируется на основе:
- Типа заполнения (из `РядыЗаполнений{XXX}`)
- Размеров (ширина × высота)
- Особенностей конструкции (термомост, тип опор, креплений)

Одинаковые по марке и размерам заполнения группируются в спецификации.

### 7. Размеры элементов конструкции

| Элемент | Формула расчёта |
|---------|----------------|
| Ригель | расстояние между стойками − 20 мм |
| Заполнение (ширина) | расстояние между стойками − 25 мм |
| Заполнение (высота) | расстояние между ригелями − 25 мм |
| Проставка ригеля | длина ригеля |
| Проставка на стойке | расстояние между ригелями − 51 мм |

### 8. Заполнения типа "Межэтажка"
На плоскостях **(5-1)-(5-6)** и **(5-6)-(5-1)** заполнения типа **Межэтажка** идут от первой до последней стойки изделия, перекрывая собой межлежащие стойки изделия.

**Столбец `Продлить`** предназначен для обозначения таких заполнений.

### 9. Ряд заполнений СН
**СН** — ряд заполнений между изделиями.

### 10. Связывающие ригели
Между изделиями есть связывающие ригели с укороченными прижимками, но **без крышек ригеля**.

### 11. Принадлежность заполнений изделиям
На плоскостях (5-1)-(5-6) и (5-6)-(5-1):
- Изделию принадлежат заполнения **справа от правой стойки** и **сверху от верхнего ригеля**

```
РядыРигелей{XXX} (основная таблица)
│
├── ТипРядовСтоек (int) → РядыСтоек{XXX}.тип
```
