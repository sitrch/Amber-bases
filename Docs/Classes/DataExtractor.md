# Класс DataExtractor

## Общая информация
- **Полное имя:** `AmberBases.Facade.DataExtractor`
- **Описание:** Статический класс для извлечения данных о рядах ригелей из загруженных Excel-файлов

## Методы

### GetFloor
Извлекает данные о рядах ригелей для заданного этажа и плоскости.

```csharp
public static CFloor GetFloor(DataSet dataSet, string плоскость, int этаж)
```

#### Параметры
| Параметр | Тип | Описание |
|----------|-----|----------|
| `dataSet` | `DataSet` | DataSet с загруженными данными из Excel |
| `плоскость` | `string` | Идентификатор плоскости (например, "(5-1)-(5-6)") |
| `этаж` | `int` | Номер этажа |

#### Возвращаемое значение
- `CFloor` — объект, содержащий данные о рядах ригелей
- `null` — если таблица не найдена или этаж не найден

#### Примечания
- Метод автоматически определяет таблицу `РядыРигелей{плоскость}` в DataSet
- Заполняет поле `CFloor.Плоскость` переданным идентификатором плоскости

## Пример использования
```csharp
// Загружаем данные из Excel
var excelDataService = new ExcelDataService();
var dataSet = excelDataService.LoadData("Bases/_Шаблоны(5-1)-(5-6).xlsx");

// Получаем данные для плоскости "(5-1)-(5-6)" и этажа 1
var floor = DataExtractor.GetFloor(dataSet, "(5-1)-(5-6)", 1);

if (floor != null)
{
    Console.WriteLine($"Плоскость: {floor.Плоскость}");
    Console.WriteLine($"Этаж: {floor.Этаж}");
    Console.WriteLine($"Низ стойки: {floor.НизСтойки}");
}
```

## Связанные классы
- `CFloor` — структура данных о рядах ригелей
- `KeyMappingRules` — класс для работы с маппингом таблиц
