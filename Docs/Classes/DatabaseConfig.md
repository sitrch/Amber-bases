# DatabaseConfig

**Модуль:** Core  
**Расположение:** `AmberBases/Core/DatabaseConfig.cs`

## Назначение
Статический класс `DatabaseConfig` предназначен для централизованного хранения базовых настроек базы данных. Сюда входит имя файла базы данных SQLite.
Это обеспечивает единую точку конфигурации и помогает избежать "магических строк" в коде. Ранее здесь также хранились жестко заданные имена таблиц, однако они были удалены в пользу динамического чтения структуры из Excel.

## Свойства и константы
- `SqliteDatabaseFile`: Имя выходного файла базы данных для выгрузки из Excel (`AmberBases.sqlite`).
- `SqliteDictionariesDatabaseFile`: Имя файла базы данных SQLite для хранения справочников (`AmberDictionaries.sqlite`).

## Зависимости
Нет внешних зависимостей.
